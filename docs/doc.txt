Аудиторские процедуры:
Аналитическая A= Analytical procedures
Наблюдение O=Observation
Инспектирование I=Inspection
Запрос/Подтверждение E=Enquiry and confirmation
Повторное выполнение U=recalcUlation/reperformance

Документальное оформление проведения аудиторской процедуры:
1. Свободный документ (текст или расчет). Содержимое не попападет в базу
2. Прикрепленный документ, любой формат. Содержимое не попадает в базу.
3. Опросник с вариантами ответов. Результат в базе.
4. Опросник без вариантов ответов. Результат в базе, но сложнее анализировать.
5. Расчет по шаблону. Результат в базе.
6. Выборка документов и результат их проверки.

Цель проведения процедуры.
Вывод по результатам проведения процедуры.
